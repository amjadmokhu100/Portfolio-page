# Tic-Tac-Toe Game

A colorful, polished 2-player Tic-Tac-Toe game built with plain HTML, CSS, and JavaScript.

## What this project includes

- A modern neon-style game board
- 2-player turn system (`X` and `O`)
- Win and draw detection
- Winning-cell highlight
- Live scoreboard (`X`, `O`, and draws)
- `Next Round` and `New Game` buttons

## How to run the game

### Option 1 (Quickest): Open directly in browser

1. Go to this project folder.
2. Double-click `index.html`.
3. Your browser will open the game.

### Option 2: Run with VS Code Live Server

1. Open this folder in VS Code/Cursor.
2. Install the **Live Server** extension (if needed).
3. Right-click `index.html`.
4. Click **Open with Live Server**.

## How to play

1. Player `X` starts first.
2. Click any empty square to place your mark.
3. Players alternate turns (`X`, then `O`).
4. First player to make 3 in a row wins:
   - Horizontal, vertical, or diagonal
5. If all 9 squares fill up with no winner, the game is a draw.
6. Click **Next Round** to keep scores and play again.
7. Click **New Game** to reset both board and scores.
